# US Healthcare Data Analysis & Dashboard Project
## 📌 Project Overview

This project focuses on analyzing a US Healthcare dataset to uncover meaningful insights related to:

Demographic trends in medical conditions

Patient billing optimization

Hospital resource management

The entire analysis and dashboard were built using Microsoft Excel, leveraging advanced Excel features such as Pivot Tables, Pivot Charts, Slicers, and Data Cleaning techniques.

## 🎯 Problem Statements
### 1️⃣ Demographic Analysis of Medical Conditions

Objective:
Identify the most prevalent medical conditions across different demographic groups.

Key Analysis Performed:

Frequency analysis of medical conditions

Age group bucketing (mutually exclusive & collectively exhaustive)

Gender-based comparison of diseases

Identification of high-risk demographic groups

Business Impact:
Helps healthcare providers understand which age groups and genders are more prone to specific conditions.

### 2️⃣ Patient Price Optimization

Objective:
Identify billing trends and recommend ways patients can reduce healthcare costs.

Key Analysis Performed:

Average billing by medical condition

Cost comparison by admission type (Emergency vs Elective)

Trend analysis of billing over time

Identification of metrics contributing to higher hospital bills

Business Impact:
Provides insights into cost drivers and suggests strategies for reducing patient expenses.

### 3️⃣ Hospital Resource Management

Objective:
Identify patterns in admissions to improve hospital workload utilization.

Key Analysis Performed:

Time-series analysis of admissions

Monthly and seasonal admission trends

Admission type distribution analysis

Workload pattern identification

Business Impact:
Supports better staffing decisions and optimized resource allocation.

## 📈 Key Insights

Certain age groups show significantly higher prevalence of chronic conditions.

Emergency admissions consistently result in higher average billing compared to elective admissions.

Specific medical conditions contribute disproportionately to total hospital revenue.

Seasonal patterns exist in patient admissions, impacting hospital workload.
